# Programs
This directory contains files for running modes 1-3 and also files as libraries for programs.

## Competition Context
This project was created for an MCU programming competition where I achieved 1st place. The original code was written rapidly under time constraints without a proper framework, relying on AI to speed up the hardware abstraction creation. The `/refactored_v2` directory contains a post-competition refactoring that implements a non-blocking architecture, effectively showcasing an understanding of state machines and proper embedded Python patterns.
